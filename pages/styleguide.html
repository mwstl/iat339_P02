Branding

Brand Identity

Evolve

Evolve is a company that creates futuristic envisionings of buildings and architecture, similar to Zaha Hadid Architecture. They offer architectural drawings, 3D renderings, and fully detailed design packages. Incorporating sci-fi styles, Evolve's work is an interesting twist on modern architecture. Learn how we use creative processes such as imagining what materials will be available in the future and how they might be manufactured, or what sort of technological advancements could occur. Evolve offers a variety of services, including architectural drawings, 3D renderings, and fully detailed design packages.

Interactive Elements

In-text link

CSS selector:

.text-link

Sample code:

<a href="#" class="text-link">SFU Homepage</a>

Rendered element:

Clicking on the following text will lead you to the SFU Homepage.

Navigation Link

CSS selector:

.button-nav

Sample code:

<a href="#" class="button-nav">Default Button</a>

Rendered element:

Default Button

Submit/Reset Buttons

CSS selector:

.btn

Sample code:

<button type="submit" class="btn" value="Submit">Submit</button>

Text Input Field with Label

CSS selector:

.label-field

Sample code:

          
<form action="" class="label-field">

  <label for="input-name">You name</label>

  <input type="text" id="input-name" name="input-name">

</form>

Sample code:

</form>
